
# Sheet: 协议格式

APP->设备 | 请求帧头（5B） |  |  |  | 命令码（2B） | 数据长度(数据包长度 2B) | 数据包（5B） | 校验和(1B)
 | 0xAA 0x21 0x01 0x31 0x01 |  |  |  | 0xXX 0xXX | 0x00 0x04 | 见下发指令 | 帧头至数据包的算术累加和
设备->APP | 应答帧头(5B) |  |  |  | 命令码（2B） | 数据长度(数据包长度 2B) | 数据包（(2+n)B） | 校验和(1B)
 | 0xBB 0x31 0x01 0x21 0x01 |  |  |  | 0xXX 0xXX | 0x00 0xXX | 状态码（2B）+返回信息 | 帧头至数据包的算术累加和
